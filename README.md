# ALME — Essenciais que você nem sabia que precisava

Site institucional desenvolvido para uma loja de produtos essenciais do dia a dia, com foco em praticidade e utilidade.

## Tecnologias utilizadas
- HTML5
- CSS3
- (Futuro: JavaScript / Frameworks)

## Objetivo do projeto
Criar uma presença digital elegante e moderna para divulgação de produtos, mostrando ofertas e informações de contato de forma clara e responsiva. Ideal para pequenos negócios que querem destacar seus produtos online.

## Status
Projeto em andamento — melhorias visuais, responsividade e integração de novos produtos planejadas.


## Sobre o projeto
ALME é um projeto de loja de produtos essenciais, desenvolvido com foco em simplicidade, design moderno e usabilidade. O site permite que os clientes conheçam os produtos e entrem em contato diretamente com a loja.

## Recursos adicionais
- Layout responsivo para desktop e mobile
- Navegação clara com menu fixo
- Seção de produtos com cards visuais
- Seções de sobre e contato já preparadas
